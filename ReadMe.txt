Url Searching:
The code creates an app that will be executed in terminal or from the command line. The app accepts a hashtag as an argument and prints out unique urls found in the 100 most recent tweets that matched the hashtag.

Sorting
Given an array of integers, the sorting code finds all the integers that appear more than once in the array, sorted by which appears most often to least often (once)
